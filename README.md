# Syslack Multiarch
Syslack Multiarch is a NetBSD Based syslack which is it also called Syslack BSD its not just a Ubuntu Based linux distro anymore it is now having BSD

No ISO Needed!

Syslack Multiarch support many architecture from NetBSD! You can list it from https://wiki.netbsd.org/ports/

If your hardware or architecture does not support Standard Version of Syslack or if you dont like Linux Syslack Multiarch is the way!

This is a Multiarch Version Based On NetBSD for Standard https://github.com/syslacklinux/Syslack-Linux/
